# Pardis Seraji Academic Page

Static GitHub Pages-ready portfolio site for Pardis Seraji.

Open `index.html` directly in a browser, or publish the folder contents to
`pardisseraji.github.io`.

The photography carousel currently uses local placeholder artwork in
`assets/`. Replace those three files with Pardis's own photographs while keeping
the same filenames, or update the image paths in `index.html`.
